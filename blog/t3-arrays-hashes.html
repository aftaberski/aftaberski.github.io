Arrays & Hashes: What's The Deal?

NOVEMBER 14, 2014

This week we started Ruby! So I thought I would celebrate with a still from Snow White & The Seven Dwarfs. In case you missed it, that seems like a ruby Doc is holding. And his facial expression probably matches mine as I'm trying to solve a exercise in Ruby. Relevant, yeah?

Luckily, Ruby (like Doc from Snow White) is really smart, but we just need to master the ins and outs in order to get our code to do exactly what we want. For starters, we'll definitely need to be able to keep track of lots of lists. No sense in doing any work that we could easily do with code. Arrays and Hashes both let us save lists, but what's the difference exactly?

Arrays

Arrays are just basic lists, essentially. They can include strings, integers, floats, other arrays...you name it! An example array looks like this:

array = ["ruby", "diamond", "emerald, saphire]

Create An Array

There are a few ways to create an array. A couple of the most popular are array = Array.new and array = [] (which is really just an empty array)

Edit An Array

Once you create an array, you might want to edit it.

You can add new objects to an array using << or .push. For example:

array << [1, 2, 3, 4]
p array

[[1, 2, 3, 4]]

OR you can do:

array.push(1, 2, 3, 4)
p array

[1, 2, 3, 4]

Some Things You Can Do With Arrays

  • array[x]: can call up a element in an array
    • For example:
    array = [1, 2, 3, 4]
    p array[2]

    #==> will return 3

Hashes

Hashes are series of key-value pairs meaning you can match elements with their value. For example, if you want to create a list of objects at a store along with their prices, you could create a hash that looks like this: hash = [:carrots => "1.0", :candy => "1.5]". But first, how do you even...

Create A Hash

Just like arrays, there are a few ways to create a hash. A couple of the most popular are hash = Hash.new and hash = [] (just like we did with arrays!)

Edit A Hash

Now, how do you edit it?

If you just created an empty hash, you can fill your hash like this:

hash = [:carrot => 1.0, :apple => 1.5]
p hash

{:carrots=>1.0, :apple=>1.5}

OR if you want to just add a single key-value pair to that same hash:

hash[:chocolate] = 2.0
p hash

{:carrots=>1.0, :apple=>1.5, :chocolate=>2.0}

Some Things Your Can Do With Hashes

  • hash.each do | x, y |: can custom print out key-value pairs
    • For example:
    hash = [:carrots=>1.0, :apple=>1.5, :chocolate=>2.0]
    hash.each do | item, price | puts "#{item}: $#{price}"
    end
    #==> will return:
    carrots: $1.0
    apple: $1.5
    chocolate: $2.0
