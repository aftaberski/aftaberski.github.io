An Array Of Solutions To Hash It Out!

NOVEMBER 14, 2014

This week we started Ruby! So I thought I would celebrate with a still from Snow White & The Seven Dwarfs. Totally relevant, right? Luckily, Ruby (like Doc from Snow White) is really smart, but we need to know how to get her to do what we want. For starters, we'll need her to keep track of lots of lists for us. Arrays and Hashes both let us save lists, but what's the difference exactly?

Arrays

Arrays are just basic lists, essentially. They can include strings or integers. An example array looks like this:

array = ["ruby", "diamond", "emerald, saphire]

Create An Array

There are a few ways to create an array. A couple of the most popular are array = Array.new and array = [] (which is really just an empty array)

Edit An Array

Once you create an array, you might want to edit it.

You can add new objects to an array using << or .push. For example:

array << [1, 2, 3, 4]
p array

[[1, 2, 3, 4]]

OR you can do:

array.push(1, 2, 3, 4)
p array

[1, 2, 3, 4]

Array Methods

  • array[x]: can call up a element in an array
    • For example:
    array = [1, 2, 3, 4]

    p array[2]

    #==> will return 3
